## Usage parameters

```
-alpha
    if set true use alpha background
-background-color string
    background color hex value (default "#FFF")
-file-name string
    file name (default "out.png")
-font string
    font path (default "./fonts/Arial.ttf")
-font-size int
    font size in points (default 96)
-foreground-color string
    foreground color hex value (default "#000")
-rect-size int
    rectangle size (default 512)
-text string
    text to print
```
